# 量化理论概念清单（快速入门版）

---

## 一、怎么用这份清单？

- **目标标准**  
  对每个概念，至少做到：
  - 能用自己的话解释；  
  - 能举 1–2 个和数字货币相关的例子；  
  - 大致知道它在“策略设计 / 回测 / 风控”里用在哪里。

- **优先级标记**  
  - `【必】`：快速入门阶段一定要搞懂；  
  - `【进阶】`：先有印象，做项目时再深入。

---

## 二、量化理论概念清单

### 1. 量化交易整体框架

- **【必】量化交易 vs 主观交易**  
  量化：规则写在程序 / 模型里，按系统执行；主观：靠盘感和临场判断。

- **【必】量化系统四层结构**  
  数据层 → 信号 / 策略层 → 组合 / 资金管理层 → 执行 / 风控层。

- **【必】“假设驱动”思维**  
  每个策略都是：
  > “市场里存在某种可重复规律” 这个假设的编码实现。

- **【进阶】边际信息 / edge**  
  你为什么比别人多一点点有效信息，或者更好地利用信息。

---

### 2. 市场与交易机制（Crypto 视角）

- **【必】交易标的与合约类型**  
  现货、永续合约、期货的基本区别，以及各自适合什么策略。

- **【必】保证金与杠杆**  
  保证金、杠杆倍数、强平价格的基本关系，为什么高杠杆特别危险。

- **【必】交易成本**  
  手续费、点差（bid-ask spread）、滑点，这三种成本如何一步步吃掉你的 edge。

- **【必】订单类型与成交规则**  
  限价单、市价单、挂单 / 吃单；撮合原则：价格优先、时间优先。

- **【进阶】市场状态 / Regime**  
  趋势市、震荡市、极端暴跌市，对趋势 / 均值回归 / 套利策略各自意味着什么。

- **【进阶】订单簿与流动性**  
  深度、挂单密度，小币 vs 大币的执行风险差异。

---

### 3. 收益、风险与绩效（与《实战数学清单》强相关）

> 这里不推公式，只说“你要懂什么”。

- **【必】收益率（简单 / 对数）**  
  回测与比较策略表现都要统一到“收益率”视角，而不是只看价格。

- **【必】波动率**  
  收益“抖动程度”的度量，决定策略的体感风险与回撤节奏。

- **【必】风险收益权衡**  
  高收益通常伴随高波动，“更高收益”不一定等于“更好的策略”。

- **【必】最大回撤**  
  从历史高点跌到低点的最大幅度，是资金方最敏感的指标之一。

- **【必】夏普比（Sharpe Ratio）**  
  单位风险赚多少钱，用来比较不同策略的“性价比”。

- **【进阶】尾部风险与厚尾**  
  小赚 + 偶尔极端大亏的策略，回测曲线可能很漂亮，但极度危险。

---

### 4. 策略与信号的基本类型

- **【必】趋势跟随（动量策略）**  
  假设：价格“涨了还会涨、跌了还会跌”，是各种均线突破、动量指标的本质。

- **【必】均值回归**  
  假设：价格或指标围绕某个“均值”上下波动，偏离太多会有回归倾向。

- **【必】套利类策略**  
  利用价差回归 / 跨品种价差 / 资金费率等“相对价格”的错误定价赚差价。

- **【必】信号 / 因子（Signal / Factor）**  
  用来预测未来收益或风险的可量化指标，例如：动量、波动率、资金费率水平等。

- **【进阶】事件驱动策略**  
  利用特定事件（公告、上线、链上大额转账等）前后的价格反应规律。

- **【进阶】多因子组合**  
  不同信号叠加，形成更稳定的预期收益（比如动量 + 价值 + 情绪）。

---

### 5. 回测与检验的核心理念

- **【必】回测的目的**  
  用历史数据验证策略假设是否大致合理，而不是“找到一条完美的曲线”。

- **【必】样本内 / 样本外**  
  用一部分数据“设计策略”（样本内），用另一部分数据“检验泛化能力”（样本外）。

- **【必】未来函数 / 数据泄露**  
  不允许用未来信息做当前决策；不允许用全样本信息造特征再只在部分区间回测。

- **【必】过拟合（Overfitting）**  
  把历史噪音当成规律，导致实盘表现远差于回测结果。

- **【进阶】Walk-forward / 滚动回测**  
  模拟真实“只看过去、不看未来”的策略更新方式。

- **【进阶】稳健性测试**  
  参数微调、不同时间段、不同标的下，策略行为是否一致。

---

### 6. 风险管理与资金管理（理论层）

- **【必】仓位管理**  
  全仓 vs 部分仓位；固定比例、固定金额三类基本思想。

- **【必】资金曲线与回撤承受度**  
  你（和未来资金方）心理上能接受多大的最大回撤、持续多久。

- **【必】杠杆风险**  
  杠杆只是在放大收益和亏损，不能凭杠杆创造 edge；高杠杆会放大所有错误。

- **【必】风险预算（Risk Budgeting）**  
  先给整体账户定一个“风险预算”，再把风险分配到不同策略 / 标的。

- **【进阶】Kelly 公式直觉**  
  给出一个“理论最优下注比例”的参考，但实际操作要打很大的折扣。

- **【进阶】止损 / 止盈规则的利弊**  
  规则本身会改变收益分布和风险特征，需要在回测中一起评估。

---

### 7. 组合与相关性视角

- **【必】相关性（Correlation）**  
  衡量两个标的 / 策略是否同涨同跌，用于判断“多样化”的真实效果。

- **【必】分散化与组合效应**  
  多个“还行”的策略组合好，可能比一个“看起来很强”的单策略更稳。

- **【必】对冲（Hedging）**  
  用一个头寸部分抵消另一个头寸的风险，例如多 BTC、空高度相关的小币。

- **【进阶】组合优化的直觉**  
  在给定风险下最大化收益，或在给定收益下最小化波动（先理解思想，不必推公式）。

---

### 8. 预测与机器学习在量化中的角色（概念版）

- **【必】回归 vs 分类**  
  - 回归：预测未来收益 / 价格变化幅度；  
  - 分类：预测“涨 / 跌 / 高波动 / 低波动”等离散结果。

- **【必】特征（Features）与标签（Labels）**  
  用哪些历史信息（特征）来预测什么结果（标签）。

- **【必】样本外表现与过拟合**  
  模型在训练集上很好看，但在新数据上表现崩掉，就是过拟合。

- **【进阶】正则化与复杂度控制**  
  模型越复杂越容易过拟合，需要控制复杂度（例如限制特征数量）。

- **【进阶】评价指标选择**  
  回归用 MSE、MAE，分类用准确率、ROC 等，但最终都要落回“策略的收益 / 风险”维度上看。

## 三、推荐使用方式

- **配合《实战数学清单》使用**  
  数学清单负责“算得清楚”，本清单负责“想得明白、说得清楚”。

- **学习时的操作模板**  
  1. 选一个小节，比如“回测与检验”；  
  2. 让 GPT 用生活 / 交易例子给你讲一遍；  
  3. 自己写一段话 + 一个 Crypto 例子，总结这个概念；  
  4. 再让 GPT 帮你出 2–3 个判断题 / 小问题自测。

- **实战中不断回填**  
  每当你在项目里遇到新概念（某个指标、某种策略、某种风险），就把它放进对应小节，
  让这份文档慢慢变成你自己的“量化理论词典”。
