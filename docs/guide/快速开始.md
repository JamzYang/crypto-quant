## ⚡ 5分钟快速体验

### 步骤1：设置环境（首次使用）

**Windows用户**：双击运行 `setup.bat`

**或者手动执行**：
```bash
# 创建虚拟环境
python -m venv quant_env

# 激活虚拟环境
quant_env\Scripts\activate

# 安装依赖（推荐使用 requirements.txt）
pip install -r requirements.txt
```

### 步骤2：运行第一个示例

```bash
# 确保已激活虚拟环境
quant_env\Scripts\activate

# 运行数学概念演示
python src/stage0/阶段0-数学预热代码示例.py
```

你会看到：
- ✅ 4个数学概念的代码演示
- 📊 生成的可视化图表
- 💡 直观理解概率、统计的含义

### 步骤3：分析真实数据

```bash
# 运行比特币数据分析项目
python src/stage0/项目1-比特币数据分析.py
```

你会得到：
- ✅ BTC历史价格数据
- 📈 完整的统计分析报告
- 📊 专业的可视化图表

## ❓ 常见问题

### Q1: 我的Python版本要求？
**A**: Python 3.8或更高版本。检查版本：
```bash
python --version
```

### Q2: 安装依赖时出错怎么办？
**A**: 常见解决方案：
```bash
# 升级pip
pip install --upgrade pip

# 使用国内镜像（更快）
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple numpy pandas matplotlib
```

### Q3: 无法连接币安API？
**A**: 两种解决方案：
1. 使用VPN/代理
2. 先用模拟数据学习（代码会自动切换）

### Q4: 数学基础很差，能学会吗？
**A**: 完全可以！本课程设计原则：
- ✅ 只学必要的数学（够用就好）
- ✅ 代码优先（通过代码理解概念）
- ✅ 项目驱动（做中学）
- ✅ 避免复杂证明（理解直觉即可）

### Q5: 每天需要学习多久？
**A**: 建议时间分配：
- **工作日**：1-2小时
- **周末**：4-6小时
- **关键**：持续>密集，每天学习比周末突击效果好

### Q6: 学完能直接实盘赚钱吗？
**A**: ⚠️ 重要提醒：
- 量化交易不是"稳赚"的圣杯
- 需要6-12个月系统学习
- 必须从小资金开始验证
- 永远做好亏损准备

## 💪 学习心态

### ✅ 正确心态
- **稳扎稳打**：基础比速度重要
- **动手为主**：看10遍不如写1遍
- **持续迭代**：今天比昨天进步一点

- **急于求成**：想一个月赚百万

- **跳步学习**：基础不牢，地动山摇
