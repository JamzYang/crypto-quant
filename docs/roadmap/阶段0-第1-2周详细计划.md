### Day 2（周二）- 概率基础
**时间**：1.5小时

- [ ] **理论学习**（30分钟）
  - 概率的定义：频率视角
  - 期望值 = 加权平均
  - 为什么赌场总是赢？（负期望游戏）
  
- [ ] **代码实践**（60分钟）
  - 打开 `阶段0-数学预热代码示例.py`
  - 详细阅读 `coin_flip_experiment()` 函数
  - **练习1**：修改抛硬币次数为 [50, 500, 5000, 50000]
  - **练习2**：模拟抛骰子，计算平均点数是否接近3.5
  - **练习3**：模拟100次交易，每次随机赚1%或亏1%，期望收益是多少？

**知识点**：
- 概率 = 有利结果数 / 总结果数
- 大数定律：实验次数越多，频率越接近理论概率
- 期望 = Σ(结果 × 概率)

### Day 3（周三）- 期望与方差
**时间**：1.5小时

- [ ] **理论学习**（30分钟）
  - 期望：平均收益
  - 方差：波动性/风险
  - 标准差：方差的平方根（与收益同单位）
  
- [ ] **代码实践**（60分钟）
  - 研究 `returns_statistics()` 函数
  - **练习1**：对比两个策略
    - 策略A：均值0.5%, 标准差1%
    - 策略B：均值1%, 标准差5%
    - 哪个更好？为什么？
  - **练习2**：模拟1000个交易员，每人100次交易
    - 有多少人最终盈利？
    - 收益分布是什么样的？

**知识点**：
- 方差 = E[(X - μ)²]
- 标准差 = √方差
- 高收益不等于好策略（还要看风险）

  ```python
  def max_drawdown(returns):
      cumulative = (1 + returns).cumprod()
      running_max = cumulative.cummax()
      drawdown = (cumulative - running_max) / running_max
      return drawdown.min()
  ```
