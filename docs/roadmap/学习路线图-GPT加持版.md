## 阶段 1（4–6 周）：实战数学最小清单 + Python 数据分析

这一阶段直接对应你已有的《实战数学清单》，目标是：  
**能看懂、能算、能用在策略分析里**，而不是做推导。

- **阶段目标**
  - 把《实战数学清单》里的 7 大块全部跑过一遍，每一块都有**配套代码实验**。
  - 能独立读懂：收益、波动率、最大回撤、Sharpe 等指标的计算代码。

- **学习内容（建议顺序）**
  - **收益与分布**  
    - 简单收益率 / 对数收益率；直方图、经验分布。
  - **基础概率直觉 + 分布形状与尾部风险**  
    - 期望、方差；大数定律直觉；偏度、峰度、厚尾；VaR 直觉。
  - **相关性与线性回归**  
    - 协方差、相关系数；一元/多元线性回归直觉。
  - **时间序列直觉**  
    - 自相关、滞后；移动平均、滚动窗口；简单 AR(1)。
  - **风险与绩效指标**  
    - 年化收益/波动率、Sharpe、最大回撤、回撤持续时间。
  - **资金管理**  
    - 仓位与杠杆关系；Kelly 公式直觉。

- **每一小块的固定学习模式（建议模板）**
  对于《实战数学清单》中的每个小点，比如“简单收益率 & 对数收益率”：

  1. **先问 GPT 要一份“生活化解释 + 交易视角解释”**。  
  2. **让 GPT 出 3–5 道小练习题**（手算/小推理）。  
  3. **你自己先做，再让 GPT 批改并解析。**  
  4. **让 GPT 生成一个“量化小实验”代码**：  
     - 输入：某个币对的历史价格。  
     - 输出：相关图表 / 数值（例如收益直方图、均值、标准差）。  
  5. 把结果截图 + 自己的理解写进 markdown。

- **阶段实战任务示例**
  - **任务 1-1：收益分布实验**  
    - 对 2–3 个不同币（BTC/ETH+一支小币），画收益直方图、算偏度/峰度。  
    - 写结论：哪个更“厚尾”？哪种更容易暴涨暴跌？
  - **任务 1-2：相关性与 pairs 直觉**  
    - 选 2 个高度相关币对（如 BTC/ETH），计算相关系数，画散点图。  
    - 写：相关性高是否等于可以做无风险套利？为什么不是？
  - **任务 1-3：简单移动平均与波动率**  
    - 用滚动标准差来定义“高波动/低波动”区间，看看不同区间里收益特征。

- **GPT 怎么帮你**
  - **数学助教**：  
    - “按照我这个水平，用故事和生活类比讲一下‘偏度/峰度’。”  
    - “请用 5 道题帮我判断是否真正理解了 VaR 直觉，并附详细解析。”
  - **实验生成器**：  
    - “用 BTC/USDT 日线数据，帮我写一段代码：计算年化收益、波动率、Sharpe、最大回撤，并画资金曲线。”
  - **复盘搭档**：  
    - 把你实验结果 + 图表发给 GPT：“从风险和尾部角度，帮我一起解读这些图，找潜在问题。”

## 阶段 2（6–10 周）：经典策略原型 + 回测与评估

核心思想：**先搞懂 2–3 个最经典策略原型，在它们上面练“构思策略 + 检查坑 + 看报表”的能力**。

- **阶段目标**
  - 至少实现并理解 2–3 个策略原型：  
    - 趋势跟随（动量、均线突破）。  
    - 均值回归（布林带、简单 pairs）。  
    - 简单波动率策略（高波动减仓、低波动加仓）。
  - 跑通一次比较完整的回测流程：  
    - 读数据 → 生成信号 → 仓位与交易 → 加手续费/滑点 → 绩效评估。

- **学习内容**
  - **策略逻辑层面**（结合《2025-11-18》的问题）  
    - 这个策略赚谁的钱？在什么市场环境会失效？  
    - 它的主要风险/尾部风险是什么？  
    - 手续费/滑点敏感不敏感？
  - **回测原则**  
    - 避免未来函数、数据泄露。  
    - 样本内 / 样本外分割；简单 walk-forward 思想。
  - **回测框架**  
    - 用 GPT 帮你选 1 个：`backtrader` 或 `vectorbt`。  
    - 学会看基本回测结果：资金曲线、回撤、Sharpe、交易次数等。

- **阶段实战任务示例**
  - **任务 2-1：单标的均线趋势策略**  
    - 策略：短均线向上突破长均线买入，反向平仓。  
    - 在 BTC 日线和 4H 上回测，比较绩效。
  - **任务 2-2：布林带均值回归策略**  
    - 价格向上/下突破布林带时反向开仓；回到中轨平仓。  
    - 观察在震荡 vs 趋势行情中的表现差异。
  - **任务 2-3：波动率调仓策略**  
    - 用滚动标准差判断波动 regime：  
      - 高波动：减仓；低波动：加仓或不开仓。  
    - 看这个简单规则对最大回撤和 Sharpe 的影响。

- **GPT 怎么帮你**
  - **策略翻译器**：  
    - 你用中文写一个逻辑：“当 20 日均线从下往上突破 60 日均线时，全仓买入；当 20 日均线从上往下跌破 60 日均线时全平仓。”  
    - 让 GPT：  
      - 先画伪代码。  
      - 再实现为回测代码。
  - **坑位检查官**：  
    - 把 GPT 给的回测代码贴回去，要求它自查：  
      - “帮我逐条检查是否存在未来函数/数据泄露/手续费没算的情况，并修改。”  
  - **结果分析助手**：  
    - 把回测报告文本/数据表粘给 GPT：“从策略逻辑的角度，帮我分析这些结果合理吗？在哪些年份或行情下失效？可能的原因是什么？”

## 阶段 3（6–10 周）：策略组合 + 风控 + 资金管理

你开始从“写单个策略的人”转变为“管理一组策略和资金的人”，更接近“小型私募运营人”的角色。

- **阶段目标**
  - 学会用组合视角看策略：相关性、多策略对冲。  
  - 掌握基本资金管理：仓位模板、风险预算、简单 Kelly。  
  - 形成一套“风险规则”：最大回撤阈值、单策略停用条件等。

- **学习内容**
  - **策略组合**  
    - 多策略、多标的组合；组合收益和风险。  
    - 相关性用于组合降波动的直觉。
  - **风控规则设计**  
    - 单笔最大亏损限制。  
    - 单策略最大回撤止损/停用规则。  
    - 组合层面最大回撤控制（比如 20%、30% 等）。
  - **资金管理与仓位**  
    - 固定比例仓位 vs 固定金额 vs 波动率调整仓位。  
    - Kelly 公式直觉，用于给出仓位上限。

- **阶段实战任务示例**
  - **任务 3-1：两策略组合实验**  
    - 用阶段 2 中表现“风格不同”的两个策略，做等权组合。  
    - 计算组合前后：波动率、Sharpe、最大回撤变化。
  - **任务 3-2：简单风险预算**  
    - 设置一个总资金，按策略的历史最大回撤分配风险预算。  
    - 策略回撤超出某阈值时自动减仓或停用。
  - **任务 3-3：仓位模板设计**  
    - 设计“保守、标准、激进”三档仓位模板，并回测各自表现。

- **GPT 怎么帮你**
  - **组合分析器**：  
    - 给 GPT 两个策略的收益序列，让它：  
      - 计算相关性、组合收益、组合风险。  
      - 用图 + 中文解释告诉你组合效果。
  - **规则生成器**：  
    - 你用中文写一段：“当策略回撤超过 15% 且持续超过 30 天时，减半仓位；超过 25% 时停用该策略。”  
    - 让 GPT 把它变成代码规则并加进回测框架。
  - **风险报告撰写**：  
    - 要求 GPT 输出一份“给潜在投资人看的风险说明文档”，你用它来反向检视自己有没有想清楚风控。

## 阶段 4（长期）：系统化运营 & 小型量化私募思维

这一阶段你已经有一套能跑的策略组合 + 回测与监控流程，开始向“运营人”升级。

- **阶段目标**
  - 把原型系统变成“能长期运转”的系统：监控、日志、告警、简单仪表盘。  
  - 学习基本的资金方沟通、产品设计（不需要太复杂）。  
  - 设计你的“小型量化私募运营流程”。

- **学习内容**
  - **工程化与监控（10% 那一块）**
    - 定时任务：每日/每小时更新数据、跑信号。  
    - 记录日志：每笔交易、每次调仓。  
    - 简单告警：回撤突破阈值、系统中断、数据异常。
  - **产品与运营**
    - 简单收益分成模式、锁定期设定。  
    - 给出月度/季度报告：收益、最大回撤、策略变更说明。  
    - 与资金方沟通的“诚实与透明”原则。

- **GPT 怎么帮你**
  - **系统架构助手**：  
    - 你描述：“我想用一个 cron job + Python 脚本 + 简单网页仪表盘”，让 GPT 帮你画出结构图和模块分工。
  - **代码填充器**：  
    - 让 GPT 把已有回测代码改成“每天自动更新并记录交易”的脚本形式。
  - **运营文档助手**：  
    - 让 GPT 基于你的实际回测/实盘结果，协助撰写：  
      - 策略说明书  
      - 风险揭示书  
      - 月度绩效简报
